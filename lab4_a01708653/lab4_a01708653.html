  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Java es un lenguaje con programación orientada a objetos (OOP), mientras que JavaScript es escrito completamente en texto y necesita ser interpretado, mientras que Java debe de ser compilado. Java se usa para crear aplicaciones en dispositivos o navegadores mientras que JavaScript se usa en documentos HTML y navegadores. Se utilizan diferentes complementos para Java y JavaScript.Java es de escritura fuerte, quiere decir que si se define un objeto de un tipo particular o crea una variable de un tipo primitivo establecido, esa variable se bloquea en su identidad. Si hay discrepancias, no puede ejecutar su código hasta que solucione el problema. Esta restricción impone buenos hábitos de codificación y requiere que las intenciones se alineen a lo largo de cualquier proyecto Java.Mientras que JavaScript es de escritura dinámica, quiere decir que permite crear variables, estructuras de datos, objetos, funciones y arreglos en línea y darles ligar cuando las necesites.
    Por otro lado, entre las principales similitudes podemos encontrar que los dos pueden coder en un navegador, las dos se puede correr en un servidor, las dos tienen librerías y frameworks.
