Preguntas de tarea

  1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?
    La recomendación es de que como el CSS es un cascading style sheets, si escribes un mismo atributo para una clase y para un selector, el que se haya puesto al último va a ser el que se sobreescriba, pues va haciendo este movimiento en cascada, pero si quieres que la regla de un selector múltiple quede como el que se va a aplicar y no quieres que se sobreescriba, tienes que poner !important, por lo que es importante priorizar primero para ver las aplicaciones del !important.
  2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?
    Primeramente, hay que asegurarnos que el texto se pueda leer bien con la imagen, además de que este tenga un color homogéneo para que no hayan problemas con partes más claras que otras o colores distintos que hagan que el texto no pueda ser legible. El fondo se de da una definición, por lo que si es en una pantalla distinta a la definida, se verá el mosaico de la imagen de forma incorrecta, hay que hacer que el fondo tenga el suficiente tamaño ya que una imagen de fondo es una imagen de mosaico repetida, por lo que el tamaño pequeño significa muchos mosaicos y va a tardar en repetir la imagen.
  3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?
    Al usar pixeles, se definen tamaños rígidos y se ignoran las configuraciones que cada usuario pueda tener en su navegador, es por ello que se recomindan usar porcentajes, donde se usa un ancho total del 100% y se adapta para que sea totalmente visible. Pt es puntos, cada punto es 0.35 milímetros, así que es útil cuando tienes medidas en un sistema de medidas que utilizas actualmente, pero de igual manera, no sería adaptable en todas las configuraciones.
  4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?
    Porque el navegador va a tener un número menor de solicitudes por leer, por lo que la carga de la página será rápida.
